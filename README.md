# 基于JSP的校园在线点餐系统

## 项目简介
校园点餐系统是基于Java Web 、JSP页面技术、Mysql数据库的在线点餐系统。系统分点餐前台系统和管理员后台管理。学生可以在前台点餐系统点餐，后台管理员可以看到点的餐品，还可以对用户、订单、菜品进行管理。本系统包含配套的毕业设计任务书、中期检查单、配套毕业论文、答辩PPT.
